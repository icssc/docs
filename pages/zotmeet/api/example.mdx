# Example API Documentation Structure

## Overview

The following is to assist those who want to edit or implement their own documentation for the ZotMeet API. The template can be found below.

<Tabs items={['Display', 'Raw Code']} defaultIndex="0">
    <Tabs.Tab>
        # Get User Information

        ------------------------------------------------------------------------------------------
        ### Get Users

        <details>
            <summary>
                `GET`
                `/users/{id}`
                `(gets user matching id)`
            </summary>

            ##### Parameters
            > | name      |  type     | data type               | description                                                           |
            > |-----------|-----------|-------------------------|-----------------------------------------------------------------------|
            > | id      |  required | uuid / string  | N/A  |

            ##### Responses
            <Tabs items={["201", "400"]}>
                <Tabs.Tab>
                    > Content-Type: `application/json`
                    > | name       | data type               | description                                                           |
                    > |------------|-------------------------|----|
                    > | id         |  uuid / string  | N/A   | N/A  |
                    > | username   | string  | N/A  |
                    > | email      | string   | N/A  |
                    > | created_at | string   | N/A  |
                </Tabs.Tab>

                <Tabs.Tab>
                    > Content-Type: `application/json`
                    > | name       | data type               | description                                                           |
                    > |------------|-------------------------|----|
                    > | code         | string  | "400"  |
                    > | message   | string  | "Bad Request"  |

                </Tabs.Tab>
            </Tabs>

            ##### Example Request
            ```js
            const id = "0000-0000-0000-0000"
            const res = await fetch(`api/users/${id}`,
            {
                method: 'GET',
                headers: { "Content-Type": "application/json"}
            })

            user = res.json()
            ```

        </details>
        ------------------------------------------------------------------------------------------
    </Tabs.Tab>
