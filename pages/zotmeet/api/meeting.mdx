## Get Multiple Meetings

<details>
    <summary>
        `GET`
        `/meetings`
    </summary>

    ##### Parameters
    > | name       | type               | description
    > |------------|-------------------------|------|
    > | id         | string  | optional |
    > | title         | string  | optional |
    > | dates         | Date[]  | optional |
    > | description         | string  | optional |
    > | location         | string  | optional |
    > | from_time         | string  | optional |
    > | to_time         | string  | optional |
    > | group_id         | string  | optional |
    > | host_id         | string  | optional |

    ##### Responses
    <Tabs items={["200", "400"]}>

            ```json copy
            {
            	"meetings": [
            		{

            				"id": "string",
            ```
